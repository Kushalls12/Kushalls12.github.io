HTML Structure

This template is composed of a fixed header and six different sections for content.

Site Header

The site header is divided into two parts, the top header and the main header.

The top header contains the email address, phone number and the social media links.

 

The main header contains the site logo, navigation menu and the book now button.

Each navigation menu link has a class of smooth-scroll which adds a smooth scrolling effect to the target section..

<li><a href="#rooms" class="smooth-scroll animated-underline">Rooms</a></li>

Sections

There are six sections in this template and they are contained by a section tag.



<section id="home"> code here... </section>

JS and Plugins



	The template uses the following JS files:



	jQuery - Javascript Library.



	aurelia.js - Custom scripts and plugins initialization.



	Google Maps API Javascript - Used for the maps found in the contact section. In order for the map to work, you need to enter your api key in the script tag.


<!-- GOOGLE MAPS API JAVASCRIPT -->
<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY_HERE" async defer></script>



	Waypoints -  Waypoints is the easiest way to trigger a function when you scroll to an element.



	Filterizr - Filterizr is a jQuery plugin that searches, sorts, shuffles and applies stunning filters over responsive galleries using CSS3 transitions. Used in the rooms section.



If you want to change or add room categories, just simply change the code below. The 'data-filter' attribute represents the category number which can be added to the 'data-category' attribute of .filtr-item



To assign room categories, add the category number to the 'data-category' attribute.





	Own Carousel - A plugin that lets you create a beautiful, responsive carousel slider. Used in the customers reviews section.



	Flaticon - Vector icons. To see all the icons included in this template, go to plugins > flaticon > font > flaticon.html.



	Vegas Background Slideshow - Vegas is a jQuery/Zepto plugin to add beautiful backgrounds and Slideshows to DOM elements. Used in the home section. To change the background images, open the aurelia.js file and look for the initHero() function, then simply change the image url of the slides object.
